If you use VS Code daily but still reach for the mouse to open files, switch tabs, or move lines — you are leaving speed on the table.

This is not another exhaustive list of every shortcut. These are the ones that actually matter, organised by how often you will use them.

## Navigation

The fastest way to open any file in your project:

- <kbd>Ctrl</kbd>+<kbd>P</kbd> — Quick Open. Start typing a filename and hit Enter. This alone eliminates most mouse usage in VS Code.
- <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd> — Command Palette. Every VS Code action lives here. If you forget a shortcut, search for the command instead.
- <kbd>Ctrl</kbd>+<kbd>G</kbd> — Go to line number. Useful when a stack trace says "error on line 47."
- <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>O</kbd> — Go to symbol in the current file. Jump to a function or class without scrolling.

## Editing

These save the most cumulative time because you use them hundreds of times a day:

- <kbd>Alt</kbd>+<kbd>Up/Down</kbd> — Move the current line up or down. No cut-and-paste needed.
- <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>K</kbd> — Delete an entire line instantly.
- <kbd>Ctrl</kbd>+<kbd>D</kbd> — Select the next occurrence of the current selection. Keep pressing to select more. This is multi-cursor editing at its simplest.
- <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>L</kbd> — Select all occurrences of the current selection at once. Rename a variable in seconds.
- <kbd>Ctrl</kbd>+<kbd>/</kbd> — Toggle comment on the selected lines.

## Multi-cursor

Multi-cursor is VS Code's superpower. Once you learn it, you will wonder how you ever edited without it:

- <kbd>Alt</kbd>+Click — Place an additional cursor anywhere.
- <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Up/Down</kbd> — Add cursors above or below the current line. Perfect for editing columnar data.
- <kbd>Ctrl</kbd>+<kbd>D</kbd> — (Repeated) adds cursors at each next match of your selection.

## Terminal

You do not need a separate terminal window:

- <kbd>Ctrl</kbd>+<kbd>`</kbd> — Toggle the integrated terminal.
- <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>`</kbd> — Create a new terminal instance.

## What to learn first

If you take away three shortcuts from this article, make them:

1. <kbd>Ctrl</kbd>+<kbd>P</kbd> — open any file
2. <kbd>Alt</kbd>+<kbd>Up/Down</kbd> — move lines
3. <kbd>Ctrl</kbd>+<kbd>D</kbd> — multi-select

These three alone will noticeably change how fast you work within a week.
